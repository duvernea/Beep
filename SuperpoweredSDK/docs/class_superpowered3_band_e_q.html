Detailed Description

Classic three-band equalizer with unique characteristics and total kills.

It doesn't allocate any internal buffers and needs just a few bytes of memory.

Parameters
bandsLow/mid/high gain. Read-write. 1.0f is "flat", 2.0f is +6db. Kill is enabled under -40 db (0.01f).
enabledTrue if the effect is enabled (processing audio). Read only. Use the enable() method to set.
