Detailed Description

IIR filters.

It doesn't allocate any internal buffers and needs just a few bytes of memory.

Parameters
frequencyCurrent frequency value. Read only.
decibelCurrent decibel value for shelving and parametric filters. Read only.
resonanceCurrent resonance value for resonant filters. Read only.
octaveCurrent octave value for bandlimited and parametric filters. Read only.
slopeCurrent slope value for shelving filters. Read only.
typeFilter type. Read only.
