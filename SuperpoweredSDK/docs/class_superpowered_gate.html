Detailed Description

Simple gate effect.

It doesn't allocate any internal buffers and needs just a few bytes of memory.

Parameters
wetLimited to >= 0.0f and <= 1.0f.
bpmLimited to >= 60.0f and <= 240.0f
beatsThe rhythm in beats to open/close the "gate". From 1/64 beats to 4 beats. (>= 0.015625f and <= 4.0f)
