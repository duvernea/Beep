typedef bool(* audioProcessingCallback_C)(void *clientdata, float **buffers, unsigned int outputChannels, unsigned int numberOfSamples, unsigned int samplerate, uint64_t hostTime)

You can have an audio processing callback with the delegate (Objective-C) or pure C. This is the pure C prototype.

Returns
Return false for no audio output (silence).
Parameters
clientDataA custom pointer your callback receives.
buffersInput-output buffers.
inputChannelsThe number of input channels.
outputChannelsThe number of output channels.
numberOfSamplesThe number of samples requested.
samplerateThe current sample rate in Hz.
hostTimeA mach timestamp, indicates when this chunk of audio will be passed to the audio output.
