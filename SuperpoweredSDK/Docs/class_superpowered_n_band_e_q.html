Detailed Description

N-band equalizer.

It doesn't allocate any internal buffers and needs just a few bytes of memory.

Parameters
decibelsGain for each frequency band in decibels. Bandwidths are automatically calculated.
enabledTrue if the effect is enabled (processing audio). Read only. Use the enable() method to set.
